
# AI Code Review Backend Setup Guide

## Prerequisites
- Node.js (v16 or higher)
- Supabase account and project
- OpenAI API key

## Step 1: Supabase Setup

1. Go to [Supabase](https://supabase.com) and create a new project
2. Create a `reviews` table in your Supabase database with the following schema:

```sql
create table reviews (
  id bigint generated by default as identity primary key,
  user_id uuid not null,
  type text not null,
  code text,
  repo_url text,
  files jsonb,
  static_analysis jsonb,
  metrics jsonb,
  ai_review jsonb,
  overall_score numeric,
  status text default 'pending',
  created_at timestamp with time zone default timezone('utc'::text, now()) not null
);

-- Enable Row Level Security (RLS)
alter table reviews enable row level security;

-- Create policies (adjust as needed for your security requirements)
create policy "Users can view their own reviews"
  on reviews for select
  using (auth.uid() = user_id);

create policy "Users can insert their own reviews"
  on reviews for insert
  with check (auth.uid() = user_id);

create policy "Users can delete their own reviews"
  on reviews for delete
  using (auth.uid() = user_id);
```

3. Get your Supabase URL and Service Role Key from Settings > API

## Step 2: Environment Variables

Copy `.env` file and update with your credentials:
```env
PORT=3001
SUPABASE_URL=your_supabase_project_url
SUPABASE_SERVICE_ROLE_KEY=your_supabase_service_role_key
OPENAI_API_KEY=your_openai_api_key
NODE_ENV=development
```

## Step 3: Install Dependencies

```bash
cd backend
npm install
```

## Step 4: Start the Server

```bash
npm start
```

The server will be running on http://localhost:3001

## API Endpoints

| Endpoint | Method | Description |
|----------|--------|-------------|
| `/api/review/code` | POST | Review pasted code |
| `/api/review/upload` | POST | Review uploaded .js/.jsx files |
| `/api/review/github` | POST | Review a public GitHub repo |
| `/api/reviews` | GET | Get all reviews for a user |
| `/api/reviews/:id` | GET | Get a specific review |
| `/api/reviews/:id` | DELETE | Delete a review |
